# **ethical-hacker-tools 🔒**

Python tools I built while learning in Class 11  
*For educational purposes only. Use responsibly.*

## *🛠️ Tools Included*
| Tool | What it does |
| --- | --- |
| `port_scanner.py` | Scan open TCP ports on any IP/domain |
| `tech_detector.py` | Detect what tech/server a website uses |
| `subdomains_finder.py` | Find subdomains of a target domain |
| `whois_lookup.py` | Get domain registration + owner info |
| `ip_geolocator.py` | Get country/city from any IP address |
| `pass_checker.py` | Check password strength |
| `hash_cracker.py` | Identify and crack hashes with wordlist |
| `gen.py` | Generate random passwords + wordlists |
