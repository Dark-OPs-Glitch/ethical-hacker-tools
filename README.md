# ethical-hacker-tools 🔒

Python tools I built while learning in Class 11

## 🛠️ Tools Included

### **1. gen.py**
**Description:** Generates random strong passwords and custom wordlists for penetration testing

**Usage:**
```bash
python gen.py
```
### **2. port_scanner**
**Description:** Scans target IP addresses for open ports 1-100. Educational tool for network security testing.

**Usage:**
```bash
python port_scanner.py
```

## **⚠️ Disclaimer**
These tools are for **educational and ethical purposes only**.  
Only test on systems you own or have explicit permission to test. Unauthorized access is illegal.
